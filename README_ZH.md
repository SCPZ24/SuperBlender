# SuperBlender

<p align="center"><strong>一个仅65行的Skill，让你的Codex从零构建Blender大作。</strong></p>

<p align="center"><a href="README.md">English</a> · <a href="README_JP.md">日本語</a> · 简体中文</p>

> SuperBlender是为[Blender MCP](https://github.com/ahujasid/blender-mcp)设计的skill。因此在使用这个skill前，你需要先给你的blender接入这个工具。

## 安装

直接告诉Codex安装需求。
```plaintext
First, verify that blender-mcp is available. Then install the skill from https://github.com/SCPZ24/SuperBlender into your skills directory.
```


## 原理

- 动手前先设计，设计前先调研
- 复用一切可复用的资源
- 用视觉验证
- 想产出大作就别吝啬token
